当其他一切都受配额限制时的应急备用。零成本 24/7 编码!

故障排除

"OAuth failed"

方案:

  • 检查网络连接
  • 尝试其他浏览器
  • 清除浏览器缓存
  • 在仪表盘重新连接

"Model not available"

方案:

  • 检查仪表盘中提供商已连接
  • 确认 OAuth token 有效
  • 必要时重新连接提供商

"Slow responses"

方案:

  • 免费层优先级较低
  • 在非高峰时段使用
  • 切换到其他免费提供商
  • 升级到低价层以提速

限制

免费层注意事项

  • 速度 - 可能慢于付费层
  • 优先级 - 高峰期优先级较低
  • 速率限制 - 可能限速(但配额无限)
  • 可用性 - 偶尔可能宕机

方案: 使用 3 层回退策略保障可靠性!
